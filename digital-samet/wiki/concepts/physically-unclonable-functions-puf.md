# Physically Unclonable Functions (PUF)

- **Source:** [[ka-hardware-security]]
- **Tags:** #hardware-security #puf #entropy #keys

## Overview
A Physically Unclonable Function (PUF) is a hardware primitive that utilizes the unique, random physical variations introduced during the semiconductor manufacturing process to provide a digital "fingerprint" of an integrated circuit.

## 1. How It Works
No two chips are identical at the transistor level due to fluctuations in doping, wire widths, and threshold voltages. A PUF circuit magnifies and digitizes these differences.
- **Challenge:** An external input to the PUF.
- **Response:** The unique output generated by the chip's physical properties for that challenge.
- **Key Property:** The mapping between challenge and response is impossible to predict or clone, even by the original manufacturer.

## 2. Main Categories

### Weak PUFs
Used primarily for **Key Generation**.
- **Characteristic:** Have a small number of challenge-response pairs (often just one).
- **Example:** **SRAM PUF**. At power-up, the random initial state of SRAM cells (determined by threshold voltage mismatches) provides the entropy.
- **Processing:** Raw PUF data is noisy (subsequent reads might differ). It requires "Secure Sketches" or Error Correcting Codes (ECC) to derive a stable, high-entropy key.

### Strong PUFs
Used primarily for **Authentication**.
- **Characteristic:** Have an exponentially large number of challenge-response pairs.
- **Example:** **Arbiter PUF**. Measures the timing difference between two identical signal paths.
- **Vulnerability:** Many strong PUFs proposed in literature are vulnerable to **Machine Learning Attacks**, where an adversary predicts the response for a new challenge based on a set of previously observed pairs.

## 3. Benefits
- **Secure Key Storage:** Eliminates the need to store keys in non-volatile memory (which is expensive and vulnerable to physical probing). The key is generated on-the-fly and "disappears" when power is off.
- **Low Cost:** Can be implemented in standard CMOS processes without extra fabrication steps.
