# Concept: Threat

## Definition
A threat is an individual, event, or action that has the capability to exploit a vulnerability. Threats are socio-technical in nature.

## Examples of Threats
- **Actors**: Hackers, disgruntled employees.
- **Events**: Poorly designed software, malicious input.
- **Processes**: Poorly articulated operational procedures.

## Relation to Vulnerability
A vulnerability is the weakness (e.g., software bug), while a threat is the action or agent that exploits it (e.g., a hacker introducing malicious input).
