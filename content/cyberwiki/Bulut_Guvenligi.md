Bulut bilişim dünyasında güvenlik kavramları, servis modelleri ve popüler platformlar.

## Bulut Servis Modelleri
- **SaaS (Software as a Service):** Yazılımın bir servis olarak sunulması (Örn: Google Drive, Microsoft 365).
- **PaaS (Platform as a Service):** Geliştiriciler için uygulama geliştirme platformu sunulması.
- **IaaS (Infrastructure as a Service):** Sunucu, depolama ve ağ gibi altyapı bileşenlerinin kiralanması (Örn: AWS EC2).

## Bulut Yayılım Modelleri
- **Public (Genel) Cloud:** Herkese açık altyapı (AWS, Azure, GCP).
- **Private (Özel) Cloud:** Sadece bir kuruma özel altyapı.
- **Hybrid Cloud:** Hem genel hem özel bulutun birleşimi.

## Temel Kavramlar
- **Serverless (Sunucusuz):** Sunucu yönetimi olmadan kod çalıştırma (Örn: AWS Lambda).
- **Infrastructure as Code (IaC):** Altyapının kodla yönetilmesi (Örn: Terraform).
- **Shared Responsibility Model:** Güvenliğin bulut sağlayıcısı ve müşteri arasında paylaşılması.

## Popüler Bulut Sağlayıcıları
- **AWS (Amazon Web Services)**
- **Azure (Microsoft)**
- **GCP (Google Cloud Platform)**

## Bulut Depolama Çözümleri
- **S3 (Amazon Simple Storage Service)**
- **Google Drive / iCloud / OneDrive / Dropbox / Box**

## English Version

Security concepts, service models and popular platforms in the world of cloud computing.

## Cloud Service Models
- **SaaS (Software as a Service):** Providing software as a service (Ex: Google Drive, Microsoft 365).
- **PaaS (Platform as a Service):** Providing an application development platform for developers.
- **IaaS (Infrastructure as a Service):** Renting infrastructure components such as server, storage and network (Ex: AWS EC2).

## Cloud Propagation Models
- **Public Cloud:** Public infrastructure (AWS, Azure, GCP).
- **Private Cloud:** Infrastructure specific to only one institution.
- **Hybrid Cloud:** A combination of both public and private cloud.

## Basic Concepts
- **Serverless:** Code execution without server management (Ex: AWS Lambda).
- **Infrastructure as Code (IaC):** Managing infrastructure with code (Ex: Terraform).
- **Shared Responsibility Model:** Sharing of security between the cloud provider and the customer.

## Popular Cloud Providers
- **AWS (Amazon Web Services)**
- **Azure (Microsoft)**
- **GCP (Google Cloud Platform)**

## Cloud Storage Solutions
- **S3 (Amazon Simple Storage Service)**
- **Google Drive / iCloud / OneDrive / Dropbox / Box**
