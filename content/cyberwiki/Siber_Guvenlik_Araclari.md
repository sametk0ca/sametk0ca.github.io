Saldırı, savunma ve analiz süreçlerinde kullanılan temel araçlar ve dağıtımlar.

## Güvenlik Dağıtımları
- **Kali Linux:** Sızma testi ve güvenlik denetimi için en popüler dağıtım.
- **ParrotOS:** Güvenlik, gizlilik ve geliştirme odaklı Debian tabanlı dağıtım.

## Keşif ve Tarama (Reconnaissance)
- **nmap:** Ağ tarama ve servis keşif aracı.
- **dig / nslookup:** DNS sorgulama araçları.
- **hping:** Paket oluşturma ve ağ test aracı.
- **curl:** Komut satırı üzerinden URL istekleri yapma.

## Analiz ve İzleme
- **Wireshark:** Grafik arayüzlü paket analiz aracı.
- **tcpdump:** Komut satırı tabanlı paket yakalama aracı.
- **Autopsy:** Dijital adli analiz (forensics) platformu.
- **FTK Imager:** Disk imajı alma ve inceleme aracı.
- **WinHex:** Hex editörü ve veri kurtarma aracı.

## İstismar ve Sonrası (Exploitation & Post-Exploitation)
- **Metasploit:** Popüler exploit framework'ü.
- **GTFOBins:** Linux sistemlerinde yetki yükseltme için kullanılan ikili dosyalar listesi.
- **LOLBAS:** Windows sistemlerinde yetki yükseltme ve saldırı için kullanılan yerleşik dosyalar.
- **WADCOMS:** Windows Active Directory saldırı teknikleri koleksiyonu.

## Çevrimiçi Analiz Servisleri
- **VirusTotal:** Dosya ve URL tarama.
- **Any.run / Joe Sandbox:** Etkileşimli zararlı yazılım analizi.
- **URLScan.io / URLVoid:** Web sitesi güvenliği tarama.
- **WHOIS:** Alan adı kayıt bilgileri sorgulama.

## English Version

Basic tools and distributions used in attack, defense and analysis processes.

## Security Deployments
- **Kali Linux:** The most popular distribution for penetration testing and security auditing.
- **ParrotOS:** Debian-based distribution focused on security, privacy and development.

## Reconnaissance and Scanning
- **nmap:** Network scanning and service discovery tool.
- **dig / nslookup:** DNS lookup tools.
- **hping:** Packet creation and network testing tool.
- **curl:** Making URL requests via command line.

## Analysis and Monitoring
- **Wireshark:** Packet analysis tool with graphical interface.
- **tcpdump:** Command line based packet capture tool.
- **Autopsy:** Digital forensics platform.
- **FTK Imager:** Disk image acquisition and analysis tool.
- **WinHex:** Hex editor and data recovery tool.

## Exploitation & Post-Exploitation
- **Metasploit:** Popular exploit framework.
- **GTFOBins:** List of binaries used for privilege escalation on Linux systems.
- **LOLBAS:** Built-in files used for privilege escalation and attack on Windows systems.
- **WADCOMS:** A collection of Windows Active Directory attack techniques.

## Online Analysis Services
- **VirusTotal:** File and URL scanning.
- **Any.run / Joe Sandbox:** Interactive malware analysis.
- **URLScan.io / URLVoid:** Website security scanning.
- **WHOIS:** Domain name registration information inquiry.
